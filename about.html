Waterdeep Campaign Visualization

This page was created to visualize D&D campaign information tracked in a separate Google spreadsheet. It is effectively a static page with Google Sheets as a backend and uses Bootstrap, JQuery, Leaflet, VisJS and Google Charts JS for interactive visualizations. Source code and static hosting uses GitHub.

Interactive Map

This project also used Python 3 to convert a large map image to optimized tiles for use in a leaflet map using the GDAL library. The interactive map leverages the Google Charts API, along with HTML, CSS, JavaScript to retrieve points of interest from a Google Sheets file and add clickable points.
